## Aufgabe 3: Git-Befehle angewendet

| Befehl | Angewendet |
|--------|------------|
| `git status` | Zustand des Arbeitsverzeichnisses geprüft |
| `git add` | Dateien zur Staging-Area hinzugefügt |
| `git commit` | Änderungen mit Nachricht gespeichert |
| `git push` | Commits zu GitHub hochgeladen |
| `git pull` | Änderungen von GitHub geholt |
| `git diff` | Unterschiede zwischen Versionen angezeigt |
| `git mv` | Frage.java umbenannt zu QuizFrage.java |
| `git rm` | Temporäre Testdatei entfernt |
